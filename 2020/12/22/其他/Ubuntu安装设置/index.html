介绍 VMware Ubuntu 安装到使用

分区

  • 方案1, 以简单为原则：
    挂载点 分区类型 分区格式 说明
    swap 逻辑分区 swap 充当虚拟内存, 大小等于内存大小
    / 主分区 ext4 安装系统和软件, 大小余下所有
  • 方案2, 资深用户：
    挂载点 分区类型 分区格式 说明
    swap 逻辑分区 swap 充当虚拟内存, 大小等于内存大小
    / 主分区 ext4 安装系统和软件, 大小 20GB 以上
    /boot 逻辑分区 ext4 引导分区, 大小 1G 左右
    /home 逻辑分区 ext4 相当于 “我的文档”, 大小为硬盘剩下的

安装 VMare Tools

VMare Tools 提供分辨率自适应, 同 win10 共享剪贴板, 直接拖动文件与 win10 交互等高级功能

  • 将VMare Tools 镜像安装到系统后复制 VMwareTools 压缩包到系统目录, 执行 一路默认设置即可

添加源和安装必备软件

  • 清华源 复制进去, 保存退出, 更新软件
  • 安装软件

    安装 sougou 输入法

  • 安装 fcitx
  • 下载安装 sougou 输入法 安装完成, 设置–>区域与语言–>管理已安装的语言, 设置为 fcitx, 然后重启即可
